## 프로젝트 소개

[![Zennbot Screen Shot][product-screenshot]](https://zennbot.net/)
젠프로덕션 음악방송에서 사용되는 챗봇 및 대시보드 어플리케이션입니다. 

구글 시트로 관리되던 사용자 포인트 정보를 이용하여 신청곡 신청 및 포인트 차감, 지급 등을 자동화합니다.
